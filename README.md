# 🏥 Hospital Management System

A responsive **Hospital Management System** developed using **HTML, CSS, and JavaScript**. This project provides a modern and user-friendly interface for managing hospital operations, including patient registration, doctor information, appointment scheduling, and authentication.

## ✨ Features

- 🔐 Login & Registration System
- 👨‍⚕️ Doctor Management
- 🧑‍🤝‍🧑 Patient Registration
- 📅 Appointment Booking
- 💊 Hospital Services
- 📋 Medical Records Management
- 💳 Billing Interface
- 📱 Responsive Design
- 🎨 Modern UI/UX
- ✅ Client-side Form Validation

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts
